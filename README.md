# BubbleGame泡泡龍遊戲
## 遊戲說明
泡泡龍是一款操作直覺、節奏輕快的益智射擊遊戲

## 基本操作

1️⃣ 瞄準

發射器通常位於畫面下方中央

2️⃣ 發射泡泡

滑鼠點擊目標位置：發射目前的泡泡

泡泡會沿直線射出，並可利用牆壁反彈

3️⃣ 消除方式

當 三顆（含）以上相同顏色的泡泡相連 時，即可消除

若消除後造成泡泡「失去支撐」，下方泡泡會一起掉落並加分

4️⃣ 失敗條件

泡泡堆積超過底線

或在限制發射次數內未完成關卡目標
